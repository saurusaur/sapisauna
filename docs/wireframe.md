# Sauna Log - Wireframe (MVP: Quick Log → Story Share)

## 화면 플로우

```
[온보딩] → [홈] → [장소선택] → [Quick Log] → [완료] → [스토리 에디터] → [공유]
```

---

## 1. 온보딩 - 타입 선택

```
┌─────────────────────────────────────┐
│                                     │
│         🧖 사우나 로그               │
│                                     │
│      나는 어떤 타입? (복수선택)        │
│                                     │
│  ┌─────────┐ ┌─────────┐ ┌─────────┐│
│  │   🛁    │ │   🔥    │ │   💆    ││
│  │         │ │         │ │         ││
│  │  목욕파  │ │ 사우너파 │ │  찜질파  ││
│  │         │ │         │ │         ││
│  │  [ ✓ ]  │ │  [ ✓ ]  │ │  [   ]  ││
│  └─────────┘ └─────────┘ └─────────┘│
│                                     │
│                                     │
│                                     │
│       ┌─────────────────────┐       │
│       │   시작하기 (2개 선택) │       │
│       └─────────────────────┘       │
│                                     │
└─────────────────────────────────────┘
```

**인터랙션**
- 카드 탭 → 선택/해제 토글
- 최소 1개 선택 시 버튼 활성화
- 선택 개수 버튼에 표시

## 컴포넌트 정의

### 슬라이더
- 범위: 1~5 (또는 온도값)
- 라벨: 양 끝에 텍스트 (예: 탁함 ↔ 맑음)
- 현재값 표시: 우측 또는 thumb 위

### 카운터 (세트 수)
- [ － ] 숫자 [ ＋ ] 형태
- 최소 1, 최대 10

### 태그/칩
- 탭하여 선택/해제
- 복수 선택 가능

### 카드
- 그림자 + 라운드 코너
- 탭 시 전체 영역 반응
