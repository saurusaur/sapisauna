# Sauna Log - Wireframe (MVP: Quick Log → Story Share)

## 화면 플로우

```
[로그인] → [온보딩 1: 닉네임] → [온보딩 2: 남탕/여탕] → [온보딩 3: 타입선택]
    → [홈] → [장소선택] → [Quick Log] → [Deep Log 넛지] → [Deep Log (선택)]
    → [완료] → [스토리 에디터] → [공유]
```

## 컴포넌트 정의

### 슬라이더
- 범위: 1~5 (또는 온도값)
- 라벨: 양 끝에 텍스트 (예: 탁함 ↔ 맑음)
- 현재값 표시: 우측 또는 thumb 위

### 카운터 (세트 수)
- [ remove ] 숫자 [ add ] 형태
- 최소 1, 최대 10

### 태그/칩
- 탭하여 선택/해제
- 복수 선택 가능

### 카드
- 그림자 + 라운드 코너
- 탭 시 전체 영역 반응

### 노렌 버튼 (남탕/여탕) - 옵션 A
- 커튼 봉 + 세로 사각형
- 세로 글씨
- 선택 시 스케일 업 + 하이라이트

### 유리문 버튼 (남탕/여탕) - 옵션 B
- 불투명 유리 느낌의 사각형
- 가로 스티커 글씨체
- 선택 시 테두리 하이라이트
