add_location_alt장소 추가

① 제목 — 단일 컬러

2톤 스택 제거. 유저 입력 제목이라 줄 분리 제어 불가 → 잉크색 한 컬러, word-break: keep-all 자연 줄바꿈. 2톤 느낌은 hue 도형·라벨이 대신함.

② 저장 완료 (방문 완료 대체)

save_place(인스타식 저장) 기반. 행 우측 북마크: 채움(레드)=저장됨, outline=탭하여 저장. 스크랩북 도장도 '저장 완료'로 — 단 북마크와 도장은 같은 상태의 중복 표현이라 최종엔 하나 선택 필요(도장이 세계관엔 더 맞고, 북마크가 행동 유도엔 더 명확).

③ SNS 링크

creator_links가 SaList 타입에 이미 정의됨(instagram/naver_blog/threads). 헤더 메타 아래 글래스 칩으로, 탭→새 탭 이동. owner 뷰에선 빈 슬롯이 '+ SNS 연결' 점선 칩.

④ 구독 버튼

스크랩북 쪽 글래스 pill 확정(h-32px, 커버 좌하단). BottomCTA 승격은 보류.

⑤ 제작자 뷰

네비: 공개 pill + share + more_vert(관리 시트) — 현 코드 그대로. 제목 옆 연필=제목/설명 편집. 행: drag_indicator로 순서(list_items.sort_order), 메모는 연필 탭→인라인 textarea(현 PlaceCard h-12 문법), 빈 메모는 점선 placeholder로 작성 유도. 맨 아래 점선 '+ 장소 추가'. 저장 북마크·구독 pill은 owner에겐 비노출.
