① 오너 편집 = 연필 단일 진입

제목 옆 edit 탭 → 기존 ListFormSheet(제목·설명·이모지 피커·hue 슬라이더)로 연결, SNS 입력 필드만 시트에 추가. 워터마크 탭 진입은 제거 — 진입점 하나로 정리. (SNS 빈 칩 '+ SNS 연결' 탭도 같은 시트의 SNS 섹션으로.)

② 오너 액션 줄 — 같은 위치·같은 단어

방문자와 동일 위치에 [구독 132] pill(채운 bookmark_add + 구독자 수, 정보성) + 공유. 네비의 share는 제거해 중복 방지(공개 pill·more_vert만 유지). 나중에 '구독 N' 탭 → 구독자 목록으로 확장 가능.

③ 심볼 + 버튼 통일

장소 저장 bookmark_heart(FILL+프라이머리 = 저장됨, place-card와 동일) · 메모 편집/빈 메모 edit · SNS photo_camera/link · 구독 bookmark_add/check · 장소 추가만 예외로 add_location_alt(장소 핀, 유저 확정). 순서 변경은 카드 길게 누르기 — 드래그 아이콘 불필요.
버튼 통일: 구독·공유·공개 모두 SNS 칩과 동일 스타일(h-24 글래스 칩, 10px bold) — 헤더 위 컨트롤이 한 가지 문법으로 정리됨.

④ 긴 시설명

시설명 flex-1 + 자연 줄바꿈(keep-all), 북마크는 우상단 고정, 동네명은 항상 둘째 줄(place-card의 주소 줄 문법과 동일). 방문자 폰 2번 카드가 예시.

⑤⑥ 기타

장소 추가 = 점선 제거, 로그 슬롯 버튼 문법(rounded-xl·틴트 bg·h-44). 상단 "SA-LIST (내 리스트)" 라벨 삭제 — 제목이 첫 요소.
