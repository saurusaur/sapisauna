① 컬러 존 플로우化 (v5 핵심)

v4까지는 존 높이 252px 고정 + 곡선 absolute → 헤더 실제 높이와 어긋나 방문자 뷰(헤더 짧음)에서 존↔카드 간격이 붕 뜨고 캡션이 곡선에 걸쳤음. v5는 hue-zone이 nav+head를 감싸는 플로우 컨테이너(내용 높이 + 14px) → 곡선 svg가 바로 아래 플로우로, 캡션·카드가 그 다음. 헤더가 길어져도(설명 2줄 등) 자동 대응.

② 간격 체계 — 균일 10px

제목·설명·메타·SNS·버튼 — 상단 요소 사이 갭 전부 10px.

③ 오너 = 구독 버튼 제거

자기 리스트를 구독할 일 없음 — 액션 줄엔 공유만. 구독자 수는 메타 줄(구독 132)로 충분. (추후 메타의 '구독 132' 탭 → 구독자 목록 확장 여지는 유지.)
