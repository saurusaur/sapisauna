## 4. 핵심 조치 필요 항목

### HIGH 우선순위
1. **그린사우나 → 현대그린사우나**: Notion 원본은 "그린사우나"인데 CSV에서 "현대그린사우나"로 변경됨. 실제 간판/Naver 등록명 확인 필요. 현대그린사우나가 맞다면 OK, 아니면 다른 시설 데이터가 섞였을 수 있음.
2. **관악사우나 → 관악24시불가마사우나**: 같은 주소(관악로 212)에 있지만 완전히 다른 상호명. 동일 시설의 공식명이 "관악24시불가마사우나"인 것인지, 같은 건물 내 다른 업체인지 확인 필요.
3. **누락 8건**: 한남사우나, 하남사우나, 삼호궁전사우나, 실로암사우나, 연지사우나, 호텔더디자이너스 서울역점, 백두산랜드, 청수탕 — 시드 등록 시 빠진 것으로 보임.

### MEDIUM 우선순위
4. **성성호수사우나 → 호수사우나**: "성성" 탈락. Naver 검색 결과가 약칭을 반환했을 가능성.
5. **일죽사우나 → 일죽목욕탕**: 업종명 변경. Naver 등록명 확인 필요.
6. **초정약수온천 → 초정약수원탕**: "온천"→"원탕" 변경. Naver 등록명 확인 필요.
