빠른 기록 = 블록 탭 (+ 어땠어 + 나만의 루틴)

👤 기록러 · 모든 트라이브 — 단순 발도장은 데이터 가치 0. 대신 "오늘 뭘 했나(사우나·온탕·냉탕…)"와 "어땠나"는 탭만으로 남기고, 온도·세트는 원하는 사람만. 마찰은 0에 가깝게, 데이터는 남게.
  • touch_app블록 탭 = 기록. 누른 순서대로 숫자가 붙어 그게 곧 루틴 순서(1건식→2냉탕→3휴식)
  • layers트라이브가 기본 블록 세트를 결정. 사우나파=건식·습식·냉탕·휴식 / 목욕파=온탕·냉탕·세신 / 찜질파=한증막·매점·수면
  • bolt빠른기록은 온도 없이 '뭐 했나'만 — 온도·세트는 ✍️상세 토글에서 (마찰 분리)
  • recommend나만의 루틴 + 추천 루틴 — 내 루틴 저장/불러오기 + 트라이브 맞춤 입문 루틴("토토노우 입문")을 써보기로 제공
출처 무브: 하루콩 블록/감정 탭 기록 + 모두의 원탭 진입 — 사-피는 트라이브 블록으로 "쉬움 × 의미 있는 데이터"를 동시에
⚖️ 진입 방식(내 제안): 기록 전 "퀵/상세" 강제 선택 화면은 마찰(결정 1개 추가). 대신 화면 상단 ⚡빠른 | ✍️상세 세그먼트 토글 + 기본값을 트라이브·직전 선택으로 기억(사우너=상세 우선, 목욕/찜질=빠른 우선). 게이트 없이 "선택" 성립. 블록↔슬라이더는 같은 컬럼(sauna_temp·totono…) 공유.
