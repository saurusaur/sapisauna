## 직접 경쟁 (사우나 도메인)

### 🇯🇵 이키타이 (Sauna-ikitai) — *가장 중요한 본보기*
- **타겟**: 일본 사우너. 트렌드는 20대가 견인
- **행동**: 사우나 검색 + **サ活(방문 기록/로그) 투고** + 시설정보 크라우드 기여
- **가치/이유**: 일본 최대 검색 DB + 기록 + **광고·유료벽 없는 신뢰 커뮤니티**
- **수익**: ① 지원형 구독 월 **370엔**(2024.2 기준 6,626명 → 월 ~245만엔) ② 굿즈 ③ 기업협찬. **광고·인기사우나 유료화·검색 유료노출 전부 거부**. 초년도부터 흑자(2년차 매출 7,000만엔·이익 1,000만엔, 자기자본)
- **기능**: 검색, サ活 로그, 크라우드 시설정보, 개인 통계, 멤버스
- **성향**: 크라우드 기여 + 조용한 기록 혼합. *신뢰* 우선
- **규모**: 월 **120만 MAU**, 등록유저 **39.2만**, 시설 14,173(전국 15,000), サ活 **1,000만건 돌파(2026.5)**. → 유저당 ~18로그, 구독전환 ~1.7%

> **SA-PI 시사점**: 사우나 기록 커뮤니티가 **광고 없이·흑자로** 성립함을 증명. 조용한 로깅이
> 규모(39만×18로그)로 작동. "지원형 구독+굿즈+협찬" = SA-PI에 가장 맞는 수익 북극성.

### 🇰🇷 사우나데이 (saunaday.co) — *한국 직접 경쟁자*
- **타겟**: 한국 사우나 애호가/웰니스 관심층
- **행동**: 지도 검색, 리뷰·별점, 체크인
- **가치/이유**: 전국 **5,000+** 발견, "동네 숨은 사우나"
- **수익**: 불명(업체 정보관리/등록 추정)
- **기능**: 네이버지도 검색, 카테고리(호텔/대형/추억), 리뷰·별점, 체크인, 매거진
- **성향**: 발견·리뷰 소비 중심
- **규모**: 시설 5,000+, 유저 수 비공개(상대적으로 작을 가능성)

> **SA-PI 시사점**: 발견·지도·리뷰는 이미 5,000개로 선점. **raw DB·지도 정면승부 = 불리.**
> 차별점은 구조화 온도/루틴 로깅 + 트라이브 + 개인가치.

### 🇰🇷 먼데이사우나 (mondaysauna.com) — *커뮤니티/브랜드* (사사사와 별개)
- **타겟**: 한국 사우나 매니아/커뮤니티
- **행동**: 카톡방 대화(추천 묻기·후기 공유), 오프라인 이벤트 참여, 콘텐츠 소비
- **가치/이유**: 소속감, 추천·후기 교환, 회복·웰니스 문화 전파
- **수익**: 불명(오프라인 이벤트·호텔 웰니스 협업·콘텐츠·굿즈 추정)
- **기능**: 카톡 커뮤니티, 사우나 검색, 매니아 추천, 크루 후기, IG/YT/Threads, 오프라인 이벤트
- **성향**: 소셜·콘텐츠·커뮤니티(vocal). 기록앱 아님
- **규모**: 비공개(카톡방+SNS 팔로워 기반)

### 🇯🇵 사레코(サレコ) — 순수 개인기록 (참고)

## 인접 기록앱 (다른 도메인, 같은 긴장구조)

### 🏃 Strava — 운동(러닝/사이클)
- **타겟**: 부유·테크리터러시 35–55 전문직(테크/금융/헬스) + 급성장 Gen Z(신규의 25%). 밀레니얼 45%·GenX 28%·GenZ 16%, 남60:여40
- **행동**: GPS 활동 로깅, **세그먼트 경쟁(KOM/QOM)**, kudos, 클럽, 챌린지
- **가치/이유**: **경쟁심**(KOM 사수)이 재방문 핵심 동인, 소셜 동기부여·책임감, 훈련 분석, 습관
- **수익**: 프리미엄 구독 **$11.99/월·$79.99/년**(루트빌더, 세그먼트 리더보드, 분석, 비콘)
- **기능**: GPS로그, 세그먼트+리더보드, 루트빌더, 히트맵(글로벌 무료/개인 유료), Relative Effort, Fitness&Freshness, Beacon, kudos/클럽/챌린지
- **성향**: 소셜·경쟁형 + "동네 산책 로깅하는 일반인"도 다수(54%가 복수 활동 기록)
- **규모**: 등록 **180–195M**, 월 +3M, 185개국, 활동 수십억(4B/2025)

### 🏋️ Hevy — 헬스(웨이트)
- **타겟**: 헬스/웨이트 트레이니
- **행동**: 운동(세트·중량) 로깅, 루틴, PR·진척 추적
- **가치/이유**: **로깅·분석이 코어**(진척 차트), 소셜은 부차
- **수익**: 프리미엄(Pro: 무제한 루틴·분석) + **Hevy Coach(B2B, 코치가 구독)**. ~$600k/월 매출(2026.2 추정)
- **기능**: 운동로그, 루틴, 진척차트, 분석, 콘텐츠 피드, **비공개 옵션**
- **성향**: *로깅 우선* — "소셜미디어 아니다" 명시. 조용한 트래커 친화
- **규모**: **200만+** 유저, 소규모 팀(2023년 13명), $240k ARR(2023)→~$600k/월(2026)

### 👟 Nike Run Club (NRC) — 러닝
- **타겟**: 러너(입문→마라톤)
- **행동**: 러닝 로깅, **가이드런(오디오 코칭)**, 챌린지, 공유
- **가치/이유**: **코칭·가이드**(콜드스타트), 정체성("Nike가 너를 러너로 만든다"), 커뮤니티 챌린지
- **수익**: **앱 자체 무료**(2020 NTC 구독 폐지). → **신발 판매·브랜드 충성**으로 회수(액티브 유저 충성도 73% vs 일반 31%)
- **기능**: GPS로그, **가이드런 ~300개**, 적응형 코칭플랜, 챌린지, 리더보드·뱃지, 라이브공유, 인앱 피드
- **성향**: 코칭 추구 + 소셜 혼합
- **규모**: 미국 iOS 월 ~40만 다운로드, Nike 6개앱 합산 1.7억+ 유저

> **SA-PI 시사점**: NRC 가이드런 ~300개 = "내 루틴 찾아가기" **콜드스타트 템플릿**이 실제
> 작동(가이드런 시 중도이탈↓)하는 증거.

### 🎬 Letterboxd — 영화
- **타겟**: 영화 팬/시네필
- **행동**: 영화 로깅·평점, 리뷰(선택), 리스트, 팔로우
- **가치/이유**: 일기·기억, 취향 정체성, 영화팬 소셜
- **수익**: 프리미엄(Pro/Patron 구독), **광고 없음**
- **기능**: 로그/다이어리, 평점, 리뷰, 리스트, 팔로우, 통계(Pro)
- **성향**: **로깅 다수** — 2024년 로그 701M : 리뷰 96.4M : 리스트 6.8M(**로그:리뷰 ≈ 7:1**), 학생 설문 14% "리뷰 아예 안 씀"
- **규모**: 활동량 거대(연 7억 로그). ⚠️ 유저수 한 출처 260만은 구버전 추정치로 보임(현재 더 큼, 미확정)

## 종합 시사점

1. **이키타이 = 비즈니스 모델 북극성.** 사우나 기록 커뮤니티가 광고 없이 흑자 가능. 조용한
   로깅이 규모로 작동(39만×18로그). 수익 = 지원형 구독 + 굿즈 + 협찬, *신뢰를 파는 게 아니라
   지키는* 모델.
2. **조용한 트래커는 주류이자 수익원.** Letterboxd 7:1, Hevy 로깅우선, 사레코 유료, 이키타이
   지원구독 → "나에게 쓸모" 축이 리텐션·매출 척추.
3. **수익 패턴 3종**: ① 지원형 구독(이키타이·Letterboxd Patron — 좋아서 냄, 유료벽 아님) ②
   프리미엄 분석(Strava·Hevy) ③ 브랜드·커머스(NRC=신발, 먼데이=이벤트/굿즈). → SA-PI엔 **①
   + ③(용품/공동구매, 마침 용품이 수요 화이트스페이스)** 조합이 가장 맞음. *커뮤니티 신뢰를
   유료벽으로 깨는 건 이키타이가 명시적으로 거부.*
4. **사우나데이와 raw DB로 싸우지 말 것.** 구조화 로깅 + 트라이브 + 개인가치(루틴핏)로 차별화.
5. **NRC 가이드런** = 콜드스타트 루틴 템플릿 정당화.
