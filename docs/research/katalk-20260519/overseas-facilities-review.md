## DB 등록 컬럼 매핑 (참고)

각 시설마다 다음 컬럼에 매핑:
- `places.country_code`: JP(20) / US(1) / HK(1) / 별도 region 코드는 입력 정책 따름
- `places.facility_type` (✅ **2026-06-01 사용자 결정 확정**):
  - `hotel-spa` (8): 아만 도쿄, 자누 도쿄, 메도우드 리조트, 홍콩 리젠트, 코노스미카, 코코로노, 미도리노카제, **fuua** (호텔/료칸/리조트 스파)
  - `public-bath` (15): 카이료유, 나미하노유, 쓰루가메유, 세이류온천, 이나리유, 마에다유, 우메유, 하기노유, 카마타 온센 (센토/공중온천) + **사우나 도쿄, 사우나스, 토토켄, 토토파, 91도 사우나** (사우나 전문 센토, Google=public_bath) + **아리마 타이코노유** (당일 온천, Google=public_bath)

  - ~~"코로"~~ 삭제: 23개 어디에도 없는 유령 항목(잔재).

## 추가 메모

1. **사우나 도쿄 vs 시부야 사우나스 vs 타카나와 사우나스**: 라이브러리에 별도 등록 필요. 타카나와점은 본 리스트에서 별도 검색 없음 (필요 시 추가 API 호출).
2. **TOTOPA 都立明治公園店**: "都立明治公園店" = 메이지공원점. 다른 지점 있을 가능성 — 본점/지점 검증 필요.
3. **이나리유 (Inariyu)**: 도쿄에 다수의 동명 센토 존재 (Takinogawa Inariyu가 가장 유명). 카톡은 "이케부쿠로" 지정 → Kamiikebukuro 위치 확정.
4. **세이류온천**: 시원하다님이 자주 가는 시설로 큐슈 가성비·숲속 노천. Nakagawa Seiryu가 가장 가능성 높지만 사용자 확인 권장.
5. **코코로노 (Hotel Furukawa)**: Google에 "心のリゾート 海の別邸 ふる川"의 영문 표시명이 "Hotel Furukawa"로 등록되어 있을 가능성. 공식 사이트(kokoronoresort.com 등)에서 별칭 확인 권장.
6. **홍콩 리젠트**: 2024년 리브랜딩으로 Intercontinental에서 Regent로 환원됨 — 시설 정보는 카톡 시점(2026-03-23) 기준.
