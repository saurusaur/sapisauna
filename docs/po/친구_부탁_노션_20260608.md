# 🧖 사-피 함께 만들기 — 도움 요청

사-피(SA-PI)는 **사우나·찜질방을 쉽고 예쁘게 기록하고, 좋은 곳을 발견·공유하는 앱**이에요.
지금 **베타 출시**를 앞두고, 영역별로 친구들 손을 빌리려 해요. 각자 맡아주면 좋은 부분을 정리했어요 🙌

> 진행 방식: 본업 사이 **서포트**라, 각 작업은 **딱 떨어지는 구현 단위**로 정리했어요. 맡을 영역 정해지면 세부는 같이 협의해요.

## 👨‍💻 개발자 — 백엔드·운영

- **지도 DB 합치기** — 전국 목욕탕 공공데이터(영업중 ~5천 곳)를 지도에 올리기. 우리가 검수한 장소(약 300곳)와 **구분해서** 노출(검수=상세 / 공공데이터=기본 정보).
- **어드민 페이지** — 지금은 장소 등록·수정이 스크립트로만 가능해요. **장소 관리 + 큐레이션(추천 리스트) 관리 UI**를 만들기.
- **프로덕트 애널리틱스** — PostHog 붙여서 **가입 → 첫 기록 → 재방문** 퍼널 측정. (지금은 유저 행동을 전혀 못 봄)

## 🎨 UX 디자이너 — 화면 구조·와이어 *(디자인만, 구현은 내가)*

- **발견(지도) 페이지** — 트라이브별 픽 필터 + 검색 흐름 개선(검색 시 지도 이동·"이 지역에서 검색"·자동완성)의 구조·와이어.
- **수집·회고 화면 전체 그림** ⭐️ *(이번 제일 크리에이티브한 파트)* — **"방문 = 수집" 컨셉**: 사우나 다녀오면 무드 젤리가 하나 쌓이고, 한 달이 모이면 나만의 컬렉션이 되고, 탭하면 그날의 추억이 펼쳐져요. **메인 메타포(유리병/앨범/뱃지…)부터 전체 흐름까지** 잡아주기. (인스피·상세는 따로 공유)
- **기록 입력(블록 탭) 인터랙션 리뷰** — 거의 완성된 화면을 디자이너 눈으로 다듬기.

> 산출물 = **바로 구현 가능한 와이어**(+ 화면별 상태: 빈 화면/로딩/에러/정상). 코드는 내가 붙여요.

## 🛠 빌더 — 게임화 기능 *(코드까지)*

- **사우나 도장판(수집)** — 방문할 때마다 무드 젤리가 쌓이고, 월간 컬렉션이 채워지고, 다 채우면 리워드·경품 추첨. (혼자 써도 쌓이는 즐겜 요소)
- **Beli식 비교 랭킹** — 다녀온 곳을 **둘씩 비교**하면 개인 점수로 자동 정렬 → "내 인생 사우나 랭킹"이 쌓이고, 친구에게 공개·비교까지.

> 둘 다 **게임처럼 만드는 재미**가 있는 파트라 골라봤어요. 디자인 에셋 없이 시작 가능하게 잡아뒀어요.
