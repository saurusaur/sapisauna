# VISION

## 프로젝트 비전

3-Click 기반 사우나/찜질방 기록 및 인스타그램 스토리 공유 PWA.
사용자 그룹(목욕파/사우너파/찜질파)별 맞춤형 로그 템플릿과 서치 가능한 데이터베이스 제공.
유저가 공개/비공개로 여러 사우나 리스트를 만들고 공유할 수 있고, 다른 유저들은 해당 리스트를 구독하여 피드에서 볼 수 있다.
