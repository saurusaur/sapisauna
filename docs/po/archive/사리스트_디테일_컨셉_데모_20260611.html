1헤더 — 곡선 컴포넌트 재사용

홈·사리스트 메인과 같은 곡선 SVG, fill만 리스트의 cover_hue 파스텔. 우하단에 그 리스트의 모티프 도형을 크게 — 메인 그리드 카드가 그대로 확대되어 열린 느낌(공유 전환 연출 가능).

22톤 타이틀 + 설명

메인 카드와 동일한 2톤 스택(잉크 1행 + hue톤 2행) → 카드 = 페이지 표지가 1:1 대응. 설명, 큐레이터 메타(@·구독수·곳수·업데이트일), 해시태그 칩.

3구독 + 공유

구독 = 잉크색 풀폭 pill(주 행동), 공유 = 옆 원형 버튼. 구독 시 "MY SHELF에 꽂혔어요" 토스트로 메인의 선반 메타포와 연결. 스크롤 내리면 구독 pill이 하단 sticky로 축소되는 패턴도 가능.

4THE LIST — A vs B

A 목차형(추천): 잡지 목차 그대로. 번호·인용선 메모라 "큐레이터가 쓴 글"의 톤이 살고 한 화면에 더 많이 보임. 행동 버튼이 적어 읽기 몰입형.

B 카드형: 시설 단위 행동(♡ 저장, 탭→시설 상세)이 분명. 현 PlaceCard와 가까워 구현 빠름. 대신 무거워지고 에디토리얼 톤은 약해짐.

절충: A 목차형 + 행 우측에 작은 ♡만 추가.
