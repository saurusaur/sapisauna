# 브랜딩 에셋 노트 (아이콘 · OG · 앱 이름)

> 2026-06-10. 아이콘/OG는 **정적 산출물**이라, 디자인 바꾸면 아래 방식으로 재생성. 원본 로고 = `public/logo/sapi-logo.svg`(빨강 #c80306 엠블럼).

## 앱 이름
- `사-피` — `public/manifest.json`(`name`/`short_name`), `src/app/layout.tsx`(`appleWebApp.title`).
- 브라우저 탭 `<title>`은 `APP.NAME`("사-피 | 우리는 사우나 신인류!") 유지(태그라인).

## 앱 아이콘 (원형, 흰 배경 + 로고 96%)
- 생성: `sharp`(devDep)로 `sapi-logo.svg`를 흰 원에 합성 → 투명 코너.
- 산출:
  - `public/icons/icon-512.png` · `icon-192.png` — PWA(원형 96%)
  - `public/icons/icon-512-maskable.png` — 안드로이드 마스커블(흰 전면배경 + 로고 68% 안전영역)
  - `src/app/icon.png`(256, 파비콘) · `src/app/apple-icon.png`(180, iOS=흰 사각+로고 82%)
  - `public/logo/sapi-logo-circle.svg` — 재사용 벡터(원형)
- 비율: 일반 96% / 마스커블 68% / 애플 82%. (원본 백업은 git 히스토리에 있음)

## OG 이미지 (`public/og-image.png`, 1200×630)
- **구조 = 홈 히어로 참고**: 레드 곡선영역(#cc1a1a) + "HELLO SA-PIEN"(Oswald Bold 이탤릭) + 서브 "사우나 신인류를 위한 기록, 공유, 발견"(설명톤) + 틸트(-3.5°) 네임카드 + 사피 로고(우측 10° 틸트, 레드 그림자).
- 카드 문구 = `좋은 사우나 방문하고 사우나템 모으기!` (= 홈 게스트카드 `STAMP_FILL_PROMPT`와 통일).
- 메타: `layout.tsx`의 `openGraph`/`twitter`(summary_large_image) + `metadataBase = https://sapisauna.vercel.app`. **도메인 바뀌면 metadataBase 수정.**
- 재생성 도구: `@resvg/resvg-js`(devDep, 폰트 명시 로드) + Oswald.
  - ⚠️ Oswald 가변폰트는 resvg가 weight 무시 → **fonttools로 static 700 인스턴싱** 후 사용:
    `python3 -m fontTools.varLib.instancer Oswald[wght].ttf wght=700 -o Oswald-700.ttf`
  - 한글은 resvg `loadSystemFonts: true`로 시스템 폰트.
