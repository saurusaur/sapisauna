## Phase 0: 디자인 토큰 교체 (Foundation) ✅ 완료

> 커밋: `d026aaa style: 디자인 오버홀 Phase 0-1`

### 0-1. globals.css — CSS 변수 교체

```
변경 전 → 변경 후:
--foreground-rgb: 68,64,60       → 42,34,34  (#2a2222)
--background-rgb: 250,249,247    → 245,242,239 (#f5f2ef)
--color-green: #2d8a6e           → --color-primary: #cc1a1a (레드)
--color-green-light: #e6f4ef     → --color-primary-light: #fde8e8
--color-orange: #e07b3c          → --color-accent: #d44040 (레드 계열)
--color-orange-light: #fef3eb    → --color-accent-light: #fde8e8
```

추가:
```css
--glass: 0 0% 100% / .35;
--glass-border: 0 0% 100% / .45;
--glass-shadow: 0 12px 40px -8px hsl(0 10% 15%/.1), ...;
--radius: .75rem;
--border: #e2deda;
--card: #faf8f6;
--muted: #e8e4e0;
--muted-foreground: #726e6e;
```

### 0-2. globals.css — 폰트 교체

```
--font-sans: 'Pretendard Variable'  → 'DM Sans', 'Noto Sans KR', sans-serif
body font-weight: (기본)             → 500
```

글래스모픽 유틸리티 클래스 추가:
```css
.glass-card { backdrop-filter: blur(12px); background: hsl(var(--glass)); ... }
.glass-input { ... }
.glass-chip { ... }
```

### 0-3. tailwind.config.ts 확장

- 새 컬러 토큰 매핑 (primary, accent, card, border, muted)
- boxShadow: glass 추가
- backdropBlur: glass 추가

### 0-4. layout.tsx — 폰트 로드 변경

- Noto Sans KR 웹폰트 추가 (Google Fonts)
- Pretendard 제거 or 폴백으로 유지

### 검증: `npm run dev` → 전체 앱 색상/폰트 일괄 변경 확인

### 커밋: `style: 디자인 토큰 교체 — 컬러·폰트·글래스모픽 기반 구축`
