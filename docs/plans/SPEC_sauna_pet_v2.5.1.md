## 0. Executive Summary

사우나 펫 V1 은 *3종의 작은 사우나 정령* 을 다마고치 미감의 픽셀풍 SVG로 구현하는 미니멀 게이미피케이션. 사우나 방문이 핵심 성장 자원이고, L3 진화 시점에 미니 퀴즈로 *진화 형태 자체가 결정* 됨 (3종 × 3 variants = 9 가능한 L3 형태). V1에 친구 시스템 + 로윌리 끼얹기 포함으로 Hydration이 사회적 활성 자원으로 작동. 베타 가입자에겐 "첫 사-피엔스" 영구 배지 + 한정 아이템. 푸시 알림과 시들음 메커닉은 V1에 없음 — totonou 톤 보호.

**예상 작업량**: 코딩 ~52h + 디자인 ~37h + 콘텐츠 ~6h = 약 **95h**. 솔로 주당 10-15h 가용 가정 7-9주. 한 달 출시 목표 시 디자인 외주 필수.

## 1. 제품 개요

### 1.1 미션 한 줄
> **사우나에 갈 때마다 함께 자라는, 당신만의 사우나 펫.**

### 1.2 기능 목적
- **리텐션**: 사우나 기록 1회마다 즉각 보상 (XP·미션 진행·외형 변화)
- **사회적 연결**: 친구의 로윌리로 펫이 회복 — 사우나의 *공유 culture*를 게임 메커닉으로
- **신규 유입**: 베타 영구 배지 + 추천인 코드로 바이럴
- **차별화**: 한국·핀란드 사우나 culture 를 펫·아이템으로 담는 사피 IP

### 1.3 성공 지표 (출시 후 4주)
| 지표 | 목표 |
|---|---|
| 사우나 펫 생성률 | 가입자 80%+ |
| W1 리텐션 (펫 있음 vs 없음) | +15%p |
| 평균 미션 완료 수 | 유저당 3+ |
| L2 도달 비율 | 40%+ |
| L3 진화 비율 | 15%+ |
| 4주 streak 달성률 | 활성 유저의 20%+ |
| 친구 1명 이상 등록 | 활성 유저의 30%+ |
| 로윌리 끼얹기 사용 | 친구 있는 유저의 50%+ |

### 1.4 비목표 (V1 범위 아님)
유료 구독 / IAP / 시즌 이벤트 / 랜덤 드롭 / 리더보드 / 실물 보상 / 푸시 알림 / 200의 법칙 / 시들음 메커닉 / 종 변경 / 인증샷 합성 / 방 꾸미기 / 사우나 다이어리 timeline

### 1.5 영구 비목표 (V1+ 어떤 버전에서도 도입하지 않음)

- **종별 스탯 차별화 / 퍼포먼스 차이** — 종 선택은 *외관/성격 preference* 이지 *전략적 최적화* 가 아님. 어떤 종을 골라도 모든 기능과 보상에 동등하게 접근. 유저가 "어떤 종이 가장 효율적일까" 고민하게 만들지 않음. *제품 원칙*.
- **사우나 펫의 죽음 / 영구 상실 메커닉** — 어떤 형태로도 펫을 잃지 않음. enthusiast 의 안식처 원칙.

> **비목표 섹션이 있는 이유**: 스코프 크립 방지 + 의식적 결정 vs 잊혀진 항목 구분. 개발 중 우선순위 흔들릴 때 명확한 기준. *영구 비목표* 는 어떤 버전에서도 다시 고민하지 않음을 명시.

## 2. 사우나 펫 컨셉 정의

### 2.1 정체성

> **사우나 펫은 한국과 핀란드 사우나 culture의 상징물에서 깨어난 작은 정령들이다.** 사우나의 열과 totonou 의 정서를 자양분 삼아 자라난다. 각자 사우나 문화의 다른 측면을 의인화한다.

세 종의 *문화적 origin*:
- **사우나돌** = 핀란드 건식 사우나 스토브 위의 돌. löyly(물을 부어 만드는 스팀)의 근원.
- **물두꺼비** = 한국 목욕탕에서 탕에 물을 채우는 돌두꺼비 동상. 사우나의 물(시원함·회복) 측면을 의인화.
- **맥반석란** = 한증막에서 천천히 익어가는 갈색 계란. 한국 찜질방의 명물.

### 2.2 3종 캐릭터 + 한 줄 소개

| 종 | 한 줄 소개 | 비주얼 키워드 | 문화 출처 |
|---|---|---|---|
| **사우나돌** | "사우나 스토브 위에서 잠들어 있던 작은 돌. 물 한 방울에 깨어난다." | 회색~갈색 둥근 돌, 작은 점 두 개의 눈, 종류에 따라 표면 질감 살짝 다름 (현무암 거친 질감) | 🇫🇮 핀란드 사우나 stones (löyly stones) |
| **물두꺼비** | "탕에 물을 채우기를 즐기는 작은 돌두꺼비. 물의 정수를 머금고 사우나 곳곳을 적신다." | **회색~짙은 회색 (돌)** 둥근 돌두꺼비 형태, 작은 눈 위쪽 위치, 살짝 미소 짓는 입. 가끔 물방울이 머리/몸 위에 맺힘. 화강암 또는 자연석 질감 | 🇰🇷 한국 목욕탕 탕 옆 물 뿜는 돌두꺼비 동상 |
| **맥반석란** | "한증막에서 천천히 익어가는 작은 계란. 완벽하게 익었을 때 비로소 빛난다." | 갈색 사우나 계란 형태, 점 두 개의 눈 + 작은 미소. 가끔 위에 김이 모락모락 | 🇰🇷 한증막 구운 계란 |

> 베이스 형태는 모두 *둥근 픽셀풍 oval* — 코스튬 슬롯이 공통으로 호환됨. 디자인 시 머리/몸/액세서리 anchor point 를 종 간 통일.

### 2.3 비주얼 — 다마고치 미감의 픽셀풍 SVG

**핵심 결정**: 다마고치는 *아트 스타일 reference* (게임 메커닉 X). 90s 픽셀 향수를 SVG 로 구현해 서버 부담 0 + 무한 확대 OK.

> 최종 스타일은 별도의 스타일 비교 시안 결과로 결정. 5개 후보 (Tamagotchi 픽셀 / 모던 픽셀 / Finch 벡터 / 카와이 / Ghibli) 중 선택.

| 요소 | 사양 |
|---|---|
| **아트 스타일** | 16~32 픽셀 그리드 기반. SVG `<rect>` 또는 단순 path 로 픽셀 표현 |
| **얼굴** | 점 두 개의 눈 + 1~2 픽셀의 입. 표정 3종: 기본 / 만족(사우나 직후) / 졸림(7일 이상 미방문 시 첫 화면) |
| **움직임** | ① 5초 주기 호흡 ② 7-15초 랜덤 깜빡임 ③ 사우나 기록 직후 1회 빛 발산 ④ 진화 풀스크린 연출 |
| **레벨 표현** | L1 → L2 → L3 진화 시 형태가 명확히 변함 (단 정체성 유지) |
| **Hydration 시각** | 100→0 감소 시 캐릭터 광택/채도 단계적 감소 (5단계). 0 도달 시 살짝 dim + 가끔 한숨 애니메이션 (압박 X, 단순 시각 indicator) |

### 2.4 베타 한정 보상 — "첫 사-피엔스"

**컨셉**: 사피의 첫 유저들에게 영구 훈장. Twitch Founders Badge 모델.

베타 가입자에게 자동 부여:
1. **코스튬 (베타 한정)**: 사피 베타 로고 작은 티셔츠 (body slot)
2. **배경 (베타 한정)**: "초석의 사우나" — 황금빛 발광 동굴 (background slot)
3. **영구 배지**: 사우나 펫 옆 작은 마크. **"첫 사-피엔스"** — 캐릭터 위 우측 상단에 표시
4. **도감 페이지**: 도감 상단에 "첫 사-피엔스" 영구 표시

베타 종료 후 가입자는 위 항목 절대 획득 불가. 마케팅 카피: *"지금 가입하면 사라지지 않는 첫 사-피엔스 흔적을 남길 수 있어요."*

## 3. 사용자 시나리오

**시나리오 A — 신규 가입**
민지는 베타로 사피 가입. 3종 중 맥반석란 선택, "맥반이" 이름. 자동으로 "첫 사-피엔스" 배지 + 베타 코스튬 받음. 다음 주 강남 사우나 방문 → 기록 → 맥반이 +240 XP, 첫 미션 "첫 발걸음" 완료.

**시나리오 B — 친구와 함께**
동훈은 친구 소영의 사우나돌이 Hydration 부족인 거 보고 *로윌리 끼얹기* (수돗물 부어주듯) → 소영 펫 +20 회복. 소영은 인앱 알림 받고 "고마워요" 답함. 양쪽 다 +5 XP 보너스.

**시나리오 C — L3 진화 모먼트**
민지의 맥반이가 1500 XP 도달. 인앱 알림 "맥반이가 진화할 준비가 됐어요". 탭하면 미니 퀴즈 3 질문 → 답변 분석 → "맥반이는 *영감란*으로 진화하려고 해요" 시안 미리보기. 민지 확정 → 풀스크린 진화 Lottie → 맥반이가 영감란 모습으로 깨어남.

**시나리오 D — 이탈 위험**
수아는 3주째 사우나 못 감. 펫 Hydration 0 — 살짝 dim. 푸시 X. 어느 날 친구가 로윌리 끼얹어주는 알림 → 인앱 켜서 다시 사우나 갈 동기.

## 5. 화면별 명세

### 5.1 사우나 펫 선택 (온보딩)

**Trigger**: 신규 가입 직후 1회만.

**구성**:
- 상단: "당신의 첫 사우나 펫을 만나보세요"
- 중앙: 3종 가로 스와이프 카드 (각 종 큰 일러스트 + 한 줄 소개 + 문화 출처)
- 하단: "선택하기" 버튼
- 선택 후 이름 입력 (8자 이내) → 확정 → 메인 뷰 이동

**베타 가입자 처리**: 종 선택 화면에서 "🌟 첫 사-피엔스 혜택 자동 적용 — 영구 배지 + 한정 코스튬/배경" 안내.

### 5.2 사우나 펫 메인 뷰

**구성** (위→아래):
1. **상단 바**: 펫 이름 / 레벨 (예: "맥반이 · Lv.2")
2. **메인 디스플레이** (화면 60%): 펫 중앙 호흡, 현재 코스튬 합성, 베타 배지(해당 시) 우측 상단
3. **상태 카드**:
   - "이번 주 사우나 1회 · 다음 진화까지 250 XP"
   - XP 진행 바
   - Hydration 게이지 ("활기" 라벨) — *친구의 로윌리로도 회복 가능* 작은 힌트
   - 4주 streak 진행 ("4주 도전: 2/4 ✅")
4. **메뉴 그리드** (2×2): 옷장 / 미션 / 친구 / 도감
5. **하단**: "사우나 다녀왔어요" 빠른 진입 버튼

**인터랙션**:
- 펫 탭 → 짧은 반응 (점프 + 미소), 하루 1회만
- 오랜만(7일+) 첫 진입 시 작은 말풍선 "오랜만이에요" 1회
- Z-order 합성: background → 펫 본체 → body → accessory → head → effect

### 5.3 옷장 (Wardrobe)

이전 버전과 동일 (탭별 슬롯, 보유/미해금 그리드, 미리보기 + 착용).

### 5.4 미션

이전 버전과 동일 (진행 중 / 완료 섹션, 자동 완료 → 수령 흐름).

## 6. 게임 메커닉

### 6.1 성장 시스템 (3 레벨, 2회 진화)

| 레벨 | 단계 | XP | 비주얼 변화 | 예상 방문 |
|---|---|---|---|---|
| 1 | 작은 X | 0 | 종별 초기 형태 | 0 |
| 2 | 종별 descriptor + X ✨첫 진화 | 200 | 종별 성장 형태 진화 | 약 2회 |
| 3 | 미니 퀴즈로 결정 ✨최종 진화 | 1500 | 종별 + 퀴즈 답변으로 variant 결정 | 약 15-20회 |

L3 도달 후: 코스튬 컬렉션 + 4주 streak 보상이 계속됨.

### 6.1.1 종별 진화 단계

| 종 | L1 (초기) | L2 (성장) | L3 (각성) — 퀴즈로 variant 결정 |
|---|---|---|---|
| **사우나돌** | 작은 사우나돌 — 회색 조약돌 | 달궈진 사우나돌 — 따뜻한 색감의 큰 돌 | **평온의 돌 / 지혜의 돌 / 신명의 돌** 중 1개 |
| **물두꺼비** | 작은 물두꺼비 — 옅은 회색 작은 돌두꺼비 모양 (돌 재질, 마른 표면) | 촉촉한 물두꺼비 — 다리·팔 있는 돌두꺼비, *물에 적셔져 살짝 짙어진 회색 + 윤기 나는 wet stone 표면* + 물방울들 | **정화꺼비 / 수호꺼비 / 복꺼비** 중 1개 (모두 돌 재질 base + variant 별 색감/장식) |
| **맥반석란** | 작은 계란 — 옅은 갈색 알 | 구운 맥반석란 — 깊은 sepia-amber, 김이 모락모락 | **무념란 / 영감란 / 황금란** 중 1개 (토게피 스타일: 윗부분 살짝 깨져 안의 빛/색감 살짝 비침) |

**중요**: 맥반석란 L3 = **토게피 스타일** 참고. 알 아랫부분 그대로, 위 일부만 깨진 형태. 안에서 따뜻한 빛 + 부드러운 색감이 살짝 비쳐 *"안에 무언가 있다"* magical curiosity 만 자극, *부화한 생명체는 절대 안 보임*.

**디자인 원칙**:
- L1 → L2 진화: 형태가 명확히 바뀜. 단 정체성 유지.
- L2 → L3 진화: 가장 극적인 변신 + 발광 + halo. *퀴즈 답변에 따라 3개 variant 중 1개로*.
- 모든 단계는 동일 anchor point 사용 — 코스튬 호환.

**총 디자인 수**: 3 (L1) + 3 (L2) + 9 (L3 variants) = **15 unique SVG**.

**진화 연출**: L2/L3 도달 시 풀스크린 Lottie (3-5초). "빛에 휩싸이며 변화" 공통 패턴 + 변신 전후 SVG 교체.

### 6.1.2 L3 미니 퀴즈 — 진화 형태를 결정하는 시스템

L3 XP 도달 시 자동 진화 *아님*. 펫이 *"진화 준비 상태"* 진입. 인앱 알림: "당신의 [펫이름]이 진화할 준비가 됐어요". 유저가 탭하면 미니 퀴즈 시작.

**퀴즈 컨셉**: *유저가 사우나를 찾는 동기* 를 3분류 (디톡스 / 사유 / 즐거움) 로 분리. 답변 패턴이 펫 진화 방향을 결정. 유저가 자기 사우나 정체성을 펫 외형에 반영시키는 셀프 캐릭터화.

#### 질문 (모든 종 공통)

**Q1. 당신이 사우나에 가는 가장 큰 이유는?**
- (a) 몸과 마음을 비우고 푹 쉬기 위해
- (b) 머리를 식히고 생각을 정리하기 위해
- (c) 친구들과 먹고 마시며 함께 시간을 보내기 위해

**Q2. 사우나에서 당신이 가장 좋아하는 순간은?**
- (a) 열기 속에서 모든 걸 잊고 땀 흘리는 시간
- (b) 멍하니 앉아 머릿속이 맑아지는 시간
- (c) 친구와 함께 도란도란 수다 떠는 시간

**Q3. 사우나 후 가장 만족스러운 순간은?**
- (a) 몸이 가볍고 개운해진 느낌
- (b) 새로운 영감이 떠오르는 그 순간
- (c) 즐거운 추억이 하나 더 생긴 기분

#### 답변 → 진화 형태 매핑

지배적 답변 (3개 중 가장 많이 선택한 옵션) 으로 L3 variant 결정:

| 지배적 답변 (동기) | 사우나돌 L3 | 물두꺼비 L3 | 맥반석란 L3 | 시각 톤 (각 종의 정체성 유지) |
|---|---|---|---|---|
| **(a) 디톡스 / 푹 휴식** | 평온의 돌 | 정화꺼비 | 무념란 | 차분, 부드러운 흰빛/푸른빛 발광, 눈 감은 평온. *물두꺼비는 옅은 회색 돌 + 물 정화의 은은한 빛* |
| **(b) 사유 / 아이디어** | 지혜의 돌 | 수호꺼비 | 영감란 | 사색적, 깊은 보라/은빛 발광, 생각하는 표정. *물두꺼비는 짙은 회색 돌 + 위엄 있는 stance* |
| **(c) 즐거움 / 컨텐츠** | 신명의 돌 | 복꺼비 | 황금란 | 화려, 따뜻한 금/주황빛 발광, 활기 있는 미소. *물두꺼비는 회색 돌 + 황금 장식/눈 (전통 복두꺼비 동상 모티프)* |

> **종 base material 유지 원칙**: 모든 L3 variant 에서 *사우나돌 = 돌 / 물두꺼비 = 돌 / 맥반석란 = 알* 의 base material 은 절대 변하지 않음. variant 차이는 *색감·발광·표정·장식* 으로만 표현. (예: 황금꺼비라 해서 황금 양서류가 되는 게 아니라, 황금 장식이 더해진 돌두꺼비)

**Tie-breaker**: 답변 동률이면 시스템이 (a) 디폴트, 단 유저가 결과 화면에서 다른 variant 선택 가능.

#### UI 흐름

1. L3 XP 도달 → 펫 메인 뷰에 알림 배지 "진화 준비됨"
2. 유저 탭 → 모달 시작: "당신의 [펫이름]이 변화의 시기를 맞이했어요. 어떤 모습으로 깨어날지 함께 정해봐요"
3. Q1 → Q2 → Q3 (한 화면당 하나, 큰 버튼 3개)
4. 답변 분석 화면: "당신의 답변을 바탕으로 [펫이름]은 → **[추천 variant 시안 미리보기]**" 1줄 설명 (variant 의 의미 함께: 예 "평온의 돌 — 모든 것을 비우고 안식을 주는 모습")
5. 분기:
   - "이 모습으로 진화시키기" → 풀스크린 Lottie → L3 진화 완료
   - "다른 진화 형태 보기" → 3개 variant 카드 + 짧은 설명 + "직접 입력 이름 (8자)" 옵션 → 유저 선택 → 진화 진행
6. 진화 후 도감에 영구 기록. **V1 변경 불가** (V2 paid 재진화 옵션 검토).

**중요 — 퍼포먼스 동등성**: 어떤 variant 로 진화하든 *XP 획득률·미션·Hydration·코스튬 접근* 모두 동일. variant 차이는 오직 *외관과 narrative 톤*. ([§1.5 영구 비목표](#15-영구-비목표-v1-어떤-버전에서도-도입하지-않음) 참조)

**개발 부담**: 모달 시리즈 + 답변 카운팅 + 9개 variant SVG + 결과 미리보기 + 분기 선택. ~6-8시간 코딩 + ~2시간 콘텐츠.

### 6.3 Hydration — V1 활성 자원

V2 계획에서 V1으로 *격상*. 친구 시스템과 함께 작동.

```yaml
hydration:
  scale: 0-100
  on_sauna_visit: +30  # 본인 사우나
  on_friend_loyly: +20  # 친구가 로윌리 끼얹어줄 때
  natural_decay: -3 per day  # 7-10일 안 가면 0
  visual_states:
    100-80: "충만" — 광택 ↑
    80-60: "괜찮음" — 기본
    60-40: "조금 마름" — 광택 ↓
    40-20: "메마름" — dim + 채도 ↓
    20-0:  "갈증" — dim + 가끔 한숨 애니메이션
  penalty: 없음  # XP 획득 정상, 미션 진행 정상
  on_friend_view: 친구 목록에서 Hydration 부족 친구 우선 표시
```

**의미**: 친구의 도움으로 회복하는 사회적 자원. 본인이 사우나 가도 회복하지만, *친구의 로윌리가 더 빠른 회복 + 정서적 가치*.

## 8. 그래픽 / 에셋 사양

### 8.1 SVG 파이프라인
- 모든 캐릭터·코스튬은 SVG 단일 파일
- 16-32 픽셀 그리드 기반 픽셀 아트
- 종별 캐릭터 SVG 는 동일 viewBox + 동일 anchor point
- Z-order 합성: background → 펫 본체 → body → accessory → head → effect

### 8.2 에셋 예산

|---|---|---|
| 캐릭터 본체 (3 L1 + 3 L2 + 9 L3) | ~8 KB | 120 KB |
| 코스튬 (19개) | ~5 KB | 95 KB |
| Lottie (사우나 후 빛 발산, L2 진화, L3 진화, 로윌리 끼얹기) | ~25 KB | 100 KB |
| **합계** | | **~315 KB** |

### 8.3 애니메이션

| 애니메이션 | 빈도 | 구현 |
|---|---|---|
| 호흡 | 항상 | CSS transform scale 5초 cycle |

## 10. 비기능 요구사항

- **성능**: 메인 뷰 로드 < 500ms. 친구 펫 보기 < 800ms.
- **오프라인**: 사우나 펫 메인 뷰 로컬 캐시. 친구 목록은 온라인 필수.
- **알림**: 푸시 V1 없음. 인앱 알림만 (친구 로윌리 받기, L3 진화 준비, 미션 완료).
- **개인정보**: 친구 펫 보기 시 본인 사우나 방문 패턴/정확한 XP는 *비공개*.
- **이름 제한**: 8자 이내. 변경은 V1 자유 (V1.5 부터 30일 쿨다운 또는 비용).
