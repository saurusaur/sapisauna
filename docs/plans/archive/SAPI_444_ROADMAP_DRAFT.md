## 2. Sapi 현재 상태 (2026-05 기준)

### 2-1. 라이브 / 기술 스택
- **상태**: 클로즈드 알파 (지인 사용, 데이터 시드 230+건 등록)
- **스택**: Next.js 14 (App Router) + TypeScript + Tailwind / Supabase (Auth·DB·RLS·Storage) / Vercel 배포 / PWA (모바일 우선)
- **외부 API**: 네이버 검색, Google Places, Google Geocoding
- **품질**: Sentry 에러 트래킹, OAuth 보안, 이용약관·개인정보처리방침 라이브
- **개발 트랙**: 2026-02 본격 시작 → 5월 현재까지 약 14주, 데일리 단위 커밋 (이키타이 수준의 깊이를 빠르게 따라잡는 중)

### 2-2. 핵심 기능 (구현 완료)
- **트라이브 시스템**: 목욕파 / 사우너파 / 찜질파 — 트라이브별 로그 템플릿·기본 루틴·UI 분기 (이키타이는 사우너 단일 페르소나, Sapi는 한국형 3-페르소나)
- **3-Click 로깅**:
  - 숏로그: 장소 → 시설/시간 → 한 줄 후기 (30초 이내)
  - 딥로그: 탕 4종(온/열/냉/급냉) 온도·점수, 사우나 건/습식, 세신·마사지, 청결도, 비용 통화 분리
- **장소 DB**: 시설유형 6종 + 탕 구분 + 시설 칩 + 타투 커버 정책 + 24시 운영 + bath_policy(남녀 분리/혼탕)
- **사-리스트(SA-LIST)**: 인스타식 저장 → 큐레이션 리스트 → 공개/비공개·구독·태그·커버 컬러/이모지·크리에이터 소셜 링크
- **추천 / 발견**:
  - 트라이브 픽 (트라이브별 추천)
  - SA-PI FEATURED (어드민 큐레이션)
  - 인기 사-리스트, 인기 태그
  - 장소 상세: 사우너 메트릭(평균 온도·세션 시간), 연관 사-리스트, 사-피엔스의 흔적
- **프로필 / 게이미피케이션 v0**:
  - XP/레벨 + 칭호 시스템 340종 (마일스톤+랜덤)
  - 프로필 아이콘 커스터마이징 (컬러+이모지)
  - 사우나 ID 카드 컨셉 (백로그)
- **공유**: 인스타 스토리용 이미지 카드 자동 생성 (HTML5 Canvas API, 트라이브 컬러·칭호·뱃지 자동 적용) + 사-피 리포트
- **히스토리 대시보드**: 캘린더 + KPI + 루틴 카드 + WEEK/MONTH 토글
- **비로그인 경험**: 홈·탐색·사-리스트 공개 (게스트 미리보기), Auth 가드 모달

### 2-3. 데이터 자산 (사사사에 의미 있는 부분)
- **장소 DB**: 한국+해외 230+건, 시설유형/탕구분/온도/타투정책/주소 정규화·도시 컬럼·좌표 보유
- **로그 데이터 스키마**: 장소별 평균 온도·평균 세션·만족도·세신비·시설별 특화 메타가 모두 정형화 → 향후 사사사 콘텐츠/리포트에 그대로 사용 가능
- **시드 리서치**: 카톡 대화 2600줄 추출, 슐렝/SPA! 어워드 등 13개 큐레이션 후보 정리 완료 → 사사사 브랜드 큐레이션과 머지 가능

| **2026-06 말** | **클로즈드 베타 v1** (사사사 커뮤니티 30~50명) | 초대 코드 + 피드백 채널, 주간 리텐션·로그/유저·리스트/유저 측정 |

### 3-3. 장기 (2027 H2~) — **커뮤니티 & 확장**

- **커뮤니티 v1**: 피드/댓글/리뷰 토론 (밀도 5,000 MAU 이상에서만 오픈)
- **오프라인 연계**: 사사사 사우나 버스 체크인·미션·NFC 스탬프, 오프라인 이벤트 → 앱 시즌 아이템 보상
- **PWA 오프라인 + 동기화**: 사우나 안에서 인터넷 안 터질 때도 기록
- **글로벌 확장**: 일본(도쿄·삿포로) 우선, 영문/일문 i18n. 이키타이와 다른 큐레이션 중심 포지셔닝 → 해외 수도권 사우너 흡수
- **B2B 데이터**: 사사사 브랜드 자산으로 시설/지자체 대상 트렌드 리포트 가능 (장소 평균 온도·세션·만족도가 이미 정형화되어 있음)

## 5. 커밋먼트 (대표님 핵심 우려 대응)

> **우려 1: 장기 커밋 가능한가?**

- **트랙레코드**: 2026-02부터 5개월간 매주 평균 5일 이상 커밋, 핵심 기능 30+ 마일스톤 달성. 풀타임 직장(코카콜라 브랜드 매니저) 병행 상태에서 입증된 페이스.
- **시간 배분**: 평일 야간 (퇴근 후 2~3h) + 주말 8~10h = 주당 약 20~25h. 베타~오픈베타 구간에는 휴가 활용 인텐스 스프린트 가능.
- **베스팅 구조 동의 (FAST 기준)**: 어드바이저 지분 2년 베스팅 + 1년 클리프 적용 가능. **2년 전체 커밋 의지 명시**.
- **Exit 시나리오 대비**: 본업 변동 시(이직·퇴사) Sapi에 풀-/하프타임 합류 옵션도 열어두고 싶음 (별도 협의).

> **우려 2: 사사사가 가져가는 가치 = 데이터·연관성**

- 모든 로그·장소·리스트 데이터는 사사사 법인이 운영하는 Supabase 프로젝트 자산으로 이전.
- 장소 DB는 사사사 웹사이트/콘텐츠 제작에도 즉시 사용 가능 (이미 정규화·도시·좌표 다 들어감).
- 사사사 브랜드와 리스트 크리에이터로 직접 결합 가능 (사-리스트 owner가 사사사 공식 아카운트 → "사사사 픽" 큐레이션 자동 생성).

> **우려 3: 단·중·장기 그림**

- 위 §3 참조. 단기는 코어 + 베타, 중기는 캐릭터/굿즈/게이미피케이션, 장기는 커뮤니티/글로벌/B2B.
